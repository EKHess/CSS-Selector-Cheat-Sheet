CSS Selectors Reference

This short Codecademy project showcases some standard HTML/CSS techniques that I've learned so far. The goal was to create a short cheat sheet for one HTML/CSS topic. I chose CSS selectors, and focused on the basics covered on the Mozzila Developers Network (MDN) documentation on the subject. To focus more on technique and less on content, the majority of the text below are excerpts from the official MDN page.

Type, Class and ID Selectors

This group includes selectors that target an HTML element such as an <h1>. It also includes selectors which target a class, or an ID.

Selector Type Description
p {}, a {}, h1 {}, etc Type/Element This selector targets an HTML element such as a(n) <p>, <a>, <h1>, etc
.myClass {} Class This selector targets HTML elements with the class class="myClass"
#mySection {} ID This group targets the HTML element with the id id="mySection"

Attribute Selectors

This group of selectors gives you different ways to select elements based on the presence of a certain attribute on an element.

Selector Type Description
a[title] {} Attribute Targets HTML elements with the attribute title
a[href="https://example.com"] {} Attribute (href) Targets HTML elements with the attribute href="https://example.com"

Pseudo-classes and Pseudo-elements

This group of selectors includes pseudo-classes, which style certain states of an element. The :hover pseudo-class for example selects an element only when it is being hovered over by the mouse pointer

It also includes pseudo-elements, which select a certain part of an element rather than the element itself. For example, ::first-line always selects the first line of text inside an element (a <p> in the below case), acting as if a <span> was wrapped around the first formatted line and then selected.

Selector Type Description
a:hover {} Pseudo-class Targets HTML <a> elements when the mouse hovers over them
p::first-letter {} Pseudo-element Targets HTML <p>elements, formatting the first letter of the element
